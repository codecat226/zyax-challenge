# Zyax frontend challenge

## A simple login form using React (react-hook-form + typescript)

### Installation and Usage

1. Clone the repo
   ```sh
   git clone https://github.com/codecat226/zyax-challenge.git
   ```
2. Install dependencies
   ```sh
   npm init
   ```
3. To run app
   ```sh
   npm start
   ```
4. To run tests
   ```sh
   npm run test
   ```
